Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get further from or closer to the equator. To accomplish this analysis, we pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude and longitude. Factors we looked at included: temperature, humidity, cloudiness, wind speed.

This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed. Feel free to poke around and check out the different plots, and thanks for visiting.
